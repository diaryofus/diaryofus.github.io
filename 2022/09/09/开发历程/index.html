主题与首页

至于主题使用方式也很简单，把个人信息什么的填写进去，首页倒计时的实现方式是用timer减去theme.date.start（这里的theme.date.start写的是"2022/08/31 00:58:09"，时间点非常严谨就是说）

<!--计时器js-->
      <script>
        function timer() {
        window.setTimeout("timer()", 1000);
        var start = new Date("<%= theme.date.start %>");
        var t = new Date().getTime() - start;
        var h = ~~(t / 1000 / 60 / 60 % 24);
        if (h < 10) {h = "0" + h;}
        var m = ~~(t / 1000 / 60 % 60);
        if (m < 10) { m = "0" + m;}
        var s = ~~(t / 1000 % 60);
       if (s < 10) {s = "0" + s;}
        document.getElementById('d').innerHTML = ~~(t / 1000 / 60 / 60 / 24);
       document.getElementById('h').innerHTML = h;
       document.getElementById('m').innerHTML = m;
       document.getElementById('s').innerHTML = s;}
        timer();
      </script> 
<!--END：计时器js-->
